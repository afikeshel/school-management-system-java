# School Management System

Java console application implementing a menu-driven school management system using object-oriented programming principles.

This project was developed as part of an Object-Oriented Programming course.

## Features

- Create and manage schools
- Manage departments
- Manage lecturers
- Assign and update chairpersons
- Display lecturer information

The system uses a menu-based interface allowing the user to create and modify school entities.

## Technologies

- Java
- Object-Oriented Programming (OOP)
- Collections
- Console-based UI

## Project Structure

The system models several entities including:

- School
- Department
- Lecturer
- Chairman

Each entity is implemented as a separate class following OOP principles such as encapsulation and modular design.

## How to Run

1. Open the project in IntelliJ IDEA
2. Run the `Main` class
3. Use the menu interface to manage school entities
